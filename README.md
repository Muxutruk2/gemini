# Gemini client

Gemini cli client written in Rust

## Usage

```
Usage: gemini [OPTIONS] <URL>

Arguments:
  <URL>

Options:
      --pager <PAGER>  [default: less] [possible values: less, more, bat, neovim]
  -h, --help           Print help
```

## Example websites

- geminispace.info
- warmedal.se
- gemini.bortzmeyer.org
